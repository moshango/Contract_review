## 🚀 快速开始命令

### 1. 配置Qwen API
编辑 `application.properties`:
```properties
qwen.api-key=sk-xxxx-xxxx-xxxx
qwen.base-url=https://dashscope.aliyuncs.com/compatible-mode/v1
qwen.model=qwen-max-latest
```

### 2. 启动应用
```bash
mvn spring-boot:run
```

### 3. 访问应用
```
http://localhost:8080/
```

### 4. 点击一键审查按钮
在规则审查界面，点击紫色的"一键Qwen审查"按钮即可开始。
