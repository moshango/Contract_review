## 🔧 项目信息

- **框架**：Spring Boot 3.5.6
- **语言**：Java 17
- **文件处理**：Apache POI / Docx4j
- **API**：RESTful HTTP (JSON)

## 🎯 核心命令

### 启动项目
```bash
mvn spring-boot:run
```

### 打包
```bash
mvn clean package
```

### 默认端口
```
http://localhost:8080
```
